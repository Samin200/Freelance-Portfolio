## About

Personal portfolio showcasing my projects, skills, and development process. Designed and built from scratch — no templates.

I'm 19, on a gap year after A-Levels, and focused on frontend development and freelance work. I use React, Tailwind CSS, and AI-assisted workflows to build clean, fast, functional websites.

| Layer | Technology |
